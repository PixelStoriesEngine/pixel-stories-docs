## Customization Options

| Setting              | Description                                                     |
| -------------------- | --------------------------------------------------------------- |
| **Background color** | Choose the fill color of the dialog box                         |
| **Border color**     | Change the outline color of the dialog box                      |
| **Size**             | Adjust the width and height of the dialog box                   |
| **Padding**          | Set spacing between text and the box edges                      |
| **Font**             | Pick the typeface used for dialog text                          |
| **Text size**        | Change how large the dialog text appears                        |
| **Text color**       | Choose the color of dialog text                                 |
| **Alignment**        | Align text left, center, or right                               |
| **Appear style**     | Control how text appears: typewriter, fade in, wavy, or instant |
